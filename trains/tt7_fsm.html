Finite State Machine

In the Digi-Comp emulator, Z0 could only set values to 0, and Z1 to 1. Here, each value can be set to 0 or 1 independently. So there are now 6 identical sections, instead of 3 pairs. The points at the moment are grayed out. Once they are interactive, more complex programs, including Nim (without the need for OR slider plates) can be programmed.
