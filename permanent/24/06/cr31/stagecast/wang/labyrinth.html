Labyrinth Mazes

Labyrinth or finger mazes have no branches, only one long path from start to finish made up of straights and bends. They use a reduced tileset of just 6 tiles.

Labyrinth set of 6 tiles

Circuit Mazes

A labyrinth maze which returns to its starting point so creating a single closed circuit (or loop). Pass-under and pass-over tiles are allowed, so the result is similar to an F1 Grand Prix racing circuit.

If a perfect maze is created using a 2-corner 'border' tileset, the result is always a circuit maze. These can be produced easily using [Stage]. Select the 2-corner 'Border' tileset and then select 'Maze'.
