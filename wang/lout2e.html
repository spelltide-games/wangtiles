Layouts of 2-edge tileset

A layout consists of a single complete tileset arranged in an array. In this case, a 4x4 array is used.
Shown below is the result of an exhaustive search of all arrangements of a 2-edge tileset where all paths are connected. Reflections and rotations have also been removed.

Tile-15 is placed in the same grid position in each layout. There are then 7 different positions for tile-0. This helps catagorise the 51 layouts.
