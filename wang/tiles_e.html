Sprite Sheets 2-edge Tilesets

All tilesets can be explored using the Stage, by selecting a '2-edge' tileset from the pull-down menu.

These images are the actual sprite sheets used by the Stage. Each 32x32 pixel tile is displayed by applying the correct horizontal (x) and vertical (y) offsets to the image.

Edge tiles are usually path type tiles and so are good for maze designs.

Shape Tiles

These tilesets have different left/right and upper/lower edge designs, in order for tiles to match. Usually, edges 'cut into' neighbouring tiles, like jigsaw pieces. As there are no clear paths, shape tilesets are not good for drawing mazes.
