2-corner

Corner Wang tiles are usually patch or terrain type tiles. Some though can show diagonal paths between tiles. These are better for drawing mazes. All tilesets are shown in the Brigid's Cross layout.

All tilesets can be explored using the Stage, by selecting a '2-corner' tileset from the pull-down menu.

These images are the actual sprite sheets used by the Stage. Each 32x32 pixel tile is displayed by applying the correct horizontal (x) and vertical (y) offsets to the image.

Flat Tilesets

These simpler 'flat' tile designs can be rotated, as real world ceramic tiles. Just 6 different tile designs are needed.
